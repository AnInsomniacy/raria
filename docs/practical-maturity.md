# Practical Maturity

`raria` is not judged by raw feature count. It is judged by whether high-value downloader behavior is implemented, testable, and honestly projected.

Current maturity priorities:

1. durable runtime semantics
2. control-plane discoverability
3. session persistence and restore
4. protocol correctness across HTTP, FTP, SFTP, Metalink, and BitTorrent
5. explicit stop-lines for incomplete BitTorrent parity

This document is intentionally narrower than “all historical aria2 behavior.”
